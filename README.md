# README

This is a simple e-market draft to generate fast Rails apps for e-shop.

Stacks used are:

Back:

  PostgreSQL
  Neon.tech (optionnal)
  Ruby (obvious)

  Auth:
    Devise
    omniauth-facebook
    omniauth-google
    bcrypt

  Payments:
    Pay
    Stripe

  Photos:
    Cloudinary

  Rich_text

Front:
  TailwindCSS
  Hotwire Turbo

Objective:
create a template from this draft to make a rails new from this template.
